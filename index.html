3D animation

Este projeto pessoal foi criado para aprender e praticar mais sobre frontend, utilizando animações 3D, CSS e consumo de APIs.

Foram utilizadas tecnologias como Three.js e a API da NASA, que permitem a criação de animações interativas e a apresentação de curiosidades sobre o espaço, incluindo as fotos do dia fornecidas pela NASA. Essa combinação de tecnologia e ciência oferece uma jornada envolvente pelo cosmos, inspirando você a explorar ainda mais!
